DuRenderer

A class used to easily manage After Effects Renderer/RenderQueue

new DuRenderer( [defaultOutputTemplate] [, defaultPostRenderAction] [, pngquant])

Constructs a new DuRenderer instance
Parameters:
Name Type Argument Default Description
defaultOutputTemplate string <optional>
 The default output module template name to use when adding comp to the queue. Empty string to let default AE OM defaultOutputTemplate.
defaultPostRenderAction PostRenderAction <optional>
 PostRenderAction.NONE The default PostRenderAction.
pngquant boolean <optional>
 false Compress PNG Sequences using pngquant after render.
Properties:
Name Type Description
defaultOutputTemplate string The default output module template name to use when adding comp to the queue. Empty string to let default AE OM defaultOutputTemplate.
defaultPostRenderAction PostRenderAction The default PostRenderAction.
pngquant boolean Compress PNG Sequences using pngquant after render.
items Array.<DuRendererItem> The DuRendererItems in the queue
